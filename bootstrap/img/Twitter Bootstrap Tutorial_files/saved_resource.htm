              Ben Russell 9 months ago

              Great introduction to Twitter Bootstrap.

              Just a note though - Bootstrap RC3 is now downloadable, but won't function correctly with this tutorial. Be sure to download V2.3.2 http://getbootstrap.com/2.3.2/
              RC3 appears to change some of the class names ie. hero-unit no longer exists, and has been renamed to jumbotron (don't ask me why! lol ). There are others too which have been either renamed or removed.

              Best bet is run through this tutorial, using V2.3.2, and then try converting it to Bootstrap RC3, and then add in some custom styling.

              Overall though an easy to follow, useful tutorial into how to use Bootstrap.

                                                Syed Fazle Rahman Mark Hallam 10 months ago

                                                Of course you can do those things. As I have mentioned in the article with the use of custom css, you can add your own personal touch to any class/element.

                                                In addition to the Twitter Bootstrap classes, you can even add your own classes like class="container homepagecontainer". Here the "container" is the class by Twitter Bootstrap and "homepagecontainer" is your own class. Later you can use this "homepagecontainer" class to add your own CSS like changing background and adding extra padding.

                                                Hope this clarifies your doubt. :)
